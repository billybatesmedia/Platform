# Sample CSV Pack

This folder contains starter CSV files for assembling a canonical V1 payload.

Use with:

- `Docs/template-system/CSV-IMPORT-RUNBOOK.md`
- `Docs/template-system/CSV-PAYLOAD-MAPPING.md`

Notes:

- Each file is UTF-8 CSV with header row.
- Keep `record_key` and `slug` stable across revisions.
- Lists use `|` delimiter.
